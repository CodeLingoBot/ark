# alloy-go
Experimental Alloy compiler in Go

## Installing
Requires Go.

	$ go get github.com/alloy-lang/alloy-go
	$ go install github.com/alloy-lang/alloy-go
	$ alloy-go

Make sure `$GOPATH/bin` is in your `$PATH`.
